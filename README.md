# settings
환경설정 스크립트

Quick setup
===========
* Download `settings` in your home directory.

```sh
~$ git clone git://github.com/DongWooYoo/settings.git
```

* run `install-settings`

```sh
~$ cd ~/settings
~/settings$ ./install-settings
```
